1.登录 [Cloudflare 仪表板](https://dash.cloudflare.com/login)。
2.从 "账户主页"，选择 "**Workers & Pages**"。
3.在**概览**中，选择你的页面项目。
4.进入 **Manage**> **Web Analytics**> 并选择 **Enable Web Analytic**。

Cloudflare 将在下一次部署时自动将 JavaScript 代码段添加到你的页面网站。
